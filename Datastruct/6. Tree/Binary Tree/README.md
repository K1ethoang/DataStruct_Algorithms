# Binary Search Tree (BST)
- Mỗi node luôn luôn có 0 - 2 nhánh con

# Duyệt cây 
- **Trước** -> NLR, NRL
- **Giữa** -> LNR, RNL
- **Sau** -> LRN, RLN

Tổn quan thì chỉ có 3 cách còn 3 cách sau là tự suy
